# NexusMind 功能清单

## 📊 项目概览

NexusMind 是一个企业级知识管理系统，采用 Spring Boot 3.3.0 + Java 21 构建，具备用户认证、工作空间管理、文件夹树形结构、文档管理等核心功能。

**技术栈：**
- 后端：Spring Boot 3.3.0、Java 21
- 安全框架：Spring Security + JWT (io.jsonwebtoken 0.12.3)
- 数据库：PostgreSQL（生产）/ H2（开发）
- ORM：Spring Data JPA / Hibernate
- 对象映射：ModelMapper
- API 文档：Swagger / OpenAPI 3.0
- 前端：React 18 + TypeScript（规划中）

## 🚀 快速开始

### 环境要求
- JDK 21+
- Maven 3.6+
- IDE（推荐 IntelliJ IDEA）
- PostgreSQL 14+（生产环境）或 H2（开发环境）

### 启动步骤
1. 配置 `application.yml` 中的数据库连接和 JWT 密钥
2. 执行 `mvn clean spring-boot:run`
3. 系统自动创建测试账户：
   - 管理员：admin/admin123
   - 普通用户：user/user123

### API 文档
- Swagger UI: http://localhost:8080/swagger-ui.html
- API Docs: http://localhost:8080/v3/api-docs

### H2 控制台（开发环境）
- URL: http://localhost:8080/h2-console
- JDBC URL: jdbc:h2:mem:nexusmind_db
- 用户名：sa
- 密码：空
