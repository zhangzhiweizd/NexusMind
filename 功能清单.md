# NexusMind 功能清单

## 📊 项目概览

NexusMind 是一个功能完整的笔记管理系统，采用 Spring Boot 3.3.0 + Java 21 构建，具备完整的 CRUD 操作、强大的文档导出功能、AI 智能摘要、富文本编辑等核心功能。

**技术栈：**
- 后端：Spring Boot 3.3.0、Java 21
- 数据库：H2（内存数据库）
- 文档处理：Apache POI 5.2.5、Flexmark、Jsoup
- AI 集成：通义千问 API
- 前端：单页面应用（SPA）

## 📤 导出功能（6 个）

### Word 导出（3 个）✅

#### 1. 导出单个笔记为 Word（.docx）✅

**支持的内容：**
- ✅ 标题（居中，18 号字体，微软雅黑）
- ✅ 摘要（斜体，12 号字体）
- ✅ 分类和标签信息
- ✅ HTML 内容解析：
  - 标题（h1-h6，不同字号）
  - 段落（左对齐，12 号宋体）
  - 列表（有序/无序）
  - 引用块（缩进，灰色斜体）
  - 代码块（等宽字体，加粗）
  - **图片（Base64/远程图片，400x300 像素）**
  - **段落内嵌图片**
- ✅ 文本格式：
  - 加粗（bold）
  - 斜体（italic）
  - 下划线（underline）
  - 删除线（strike-through）
  - 上标/下标（superscript/subscript）
- ✅ 创建时间（右对齐，10 号字体）

**技术实现：**
- Apache POI XWPF 库
- Jsoup HTML 解析
- Base64 图片解码
- 动态图片类型识别（PNG/JPEG/GIF）
- EMU 单位（9525 = 1 像素）

**API 接口：**
- `GET /api/notes/{id}/export/word` - 导出单个笔记为 Word

#### 2. 批量导出笔记为 Word（ZIP 压缩包）✅

**功能描述：**
- 将多个笔记导出为 Word 文档
- 自动打包成 ZIP 格式
- 文件名自动处理（移除特殊字符）

**API 接口：**
- `POST /api/notes/export/word` - 批量导出为 Word

## 🚀 快速开始

### 环境要求
- JDK 21+
- Maven 3.6+
- IDE（推荐 IntelliJ IDEA）

1. 配置 `application.yml` 中的 AI API Key

3. 访问 http://localhost:8080

### API 文档

- API Docs: http://localhost:8080/v3/api-docs
